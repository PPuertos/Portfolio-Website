Stock Market Portfolio Management Software
Empower your investment strategies by transforming historical data into actionable insights. This autonomous software provides a seamless experience, utilizing cutting-edge algorithms to maximize returns and minimize risks, designed for both novice and seasoned investors alike.
Value Proposition
This platform offers a comprehensive solution for portfolio management, enabling users to visualize historical trends, analyze performance, and apply predictive models for smarter decision-making. By integrating diverse data sources, it delivers a robust toolset for optimizing returns and proactively managing risks, ensuring you stay ahead in dynamic market conditions.
Features
Top-Down Analysis
Explore your portfolio with interactive, multi-level charts, from markets to individual stocks, with real-time performance data.
Performance
Monitor key metrics such as alpha, beta, VaR, CVaR and Volatility with a focus on identifying potential red flags in your investments.
Efficient Frontier
Optimize investments using the Markowitz model and Monte Carlo simulations to maximize returns and minimize risk.
Real Time Insights
Stay updated with stock news, top daily gainers, and most active stocks, directly in your dashboard.
Stock Performance Evaluation
Assess individual stocks with trend identification and essential financial metrics to gauge investment potential.
Customized Views
Easily adjust markets and timeframes to tailor your portfolio experience to your specific investment strategy.
Technologies Implemented
Python
Primary language for building the application, integrating data processing, analysis, and visualization libraries.
yfinance
Extraction of historical stock prices, company financial health data, additional business information, and popular stock metrics.
Pandas
Data management through DataFrames, enabling matrix operations and efficient data manipulation.
Scipy
Implementation of portfolio risk minimization and Sharpe ratio maximization using the optimization module.
Numpy
Calculation of basic statistical measures such as covariance, variance, and mean.
Plotly
Development of interactive visualizations using graph_objects and integration within the Dash application.
Finnhub API
Retrieval of real-time news updates for stocks in the user’s portfolio or researched companies.
Banxico API
Acquisition of current USD to MXN exchange rate for accurate financial metrics.
DBC
Application of Bootstrap classes to enhance styling and layout of the webpage.

Steps to run it yourself

01

Clone the Repository

Obtain a local copy of the project repository using the command:
git clone https://github.com/PPuertos/Stock-Market-Dashboard/stockMarketDashboardNew.git

02

Install Required Libraries

Install all necessary libraries by running:
pip install -r requirements.txt
This command ensures that all dependencies listed in 'requirements.txt' are installed.

03

Run the Main Script

Start the application by executing the main script with 'python src/app.py'. Here, 'app.py' serves as the entry point for the application.

04

Explore the Dashboard

Access the dashboard via your web browser to interact with the map, cost panels, and business intelligence plots.

05

Customize Data (Optional)

To use your own data, create a CSV file with the columns '[trailer_id, destination]' and replace the existing data file as needed.

06

Result

After completing the setup, you will be able to visualize and analyze transportation routes and associated costs, providing valuable insights into route feasibility and cost efficiency.
