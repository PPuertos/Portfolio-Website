Dashboard: Transportation Cost Logistics
Leverage data-driven insights to optimize transportation logistics. This autonomous dashboard is designed to track and analyze the costs associated with all routes for trailers, enabling the evaluation of route feasibility and enhancing business intelligence.
Value Proposition
This dashboard integrates various data sources and visualization tools to offer a comprehensive view of transportation logistics, helping to drive informed decision-making and optimize logistics operations.
Features
Visualize
Visualization of routes on an interactive map.
Analyse
Cost analysis panel by route.
Get Insights
Business intelligence plots for in-depth data insights.
Easy to use
User-friendly interface for seamless navigation.
Technologies Implemented
Programs
Python
Core Libraries
INEGIpy
Extracts route (GEOJSON format), freight and diesel cost data.
Pandas
Facilitates data manipulation and analysis.
Plotly
Creates interactive visualizations and plots. Dash module powers the web-based dashboard interface.
Follium
Displays routes on a map

Steps to run it yourself

01

Clone the Repository

Obtain a local copy of the project repository using the command:
git clone https://github.com/PPuertos/Dashboard-Logistica.git

02

Install Required Libraries

Install all necessary libraries by running:
pip install -r requirements.txt
This command ensures that all dependencies listed in 'requirements.txt' are installed.

03

Run the Main Script

Start the application by executing the main script with 'python src/app.py'. Here, 'app.py' serves as the entry point for the application.

04

Explore the Dashboard

Access the dashboard via your web browser to interact with the map, cost panels, and business intelligence plots.

05

Customize Data (Optional)

To use your own data, create a CSV file with the columns '[trailer_id, destination]' and replace the existing data file as needed.

06

Result

After completing the setup, you will be able to visualize and analyze transportation routes and associated costs, providing valuable insights into route feasibility and cost efficiency.
